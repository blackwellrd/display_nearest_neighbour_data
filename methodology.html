Methodology

Nearest Neighbours

All the above fields are scaled by centring the values by subtracting the mean of the values from each value and then scaled by dividing the centred values by the standard deviation of the centred values.

The Euclidian distance between pairs of organisations are calculated as follows, the distance between organisation p and organisation q is

$$(d, q) = \sqrt{(p_1 - q_1)^2 + (p_2 - q_2)^2 + \cdots + (p_{k} - q_{k})^2}$$

where k is the number of variables used in the matching process.

These distances are calculated for all points $$\frac{n * (n-1)}{2}$$for practices that is $$\frac{(6333 * (6333 - 1))}{2} = 20,050,278$$ and for PCNs that is $$\frac{(1271 * (1271 - 1))}{2} = 807,085$$

The top n (in the example we will use the top 10) closest organisations are used as the nearest neighbours
